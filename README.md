This project explores statistical arbitrage strategies in cryptocurrency markets, with a focus on momentum and reversal patterns. The goal is to identify market inefficiencies in crypto assets and evaluate whether these strategies can generate profitable trading signals.

The project is designed to mirror the type of research conducted at quantitative hedge funds, combining data analysis, strategy design, and performance evaluation.
